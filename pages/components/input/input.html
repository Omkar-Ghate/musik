    Input

    Used to get user input

    Textbox

    A responsive textbox can be used to enter a brief message.

    Simple Input

    Use classes input-outlined, input-filled and input-standard to get outlined input,filled input and standard input repectivaly.

    Input with Label

    We can use label with inputs. Use label class to add label to your input. Wrap the input and label in input-container.

    Input Error

    Use class input-error to show error in input.

    Input Required Validation

    Use class input-required to visually validate required input. Input first will have danger color border which will turn to safe color as you type.

    Form Validation

    Below is the form validation example. You can have error message and check passed message on field level.
